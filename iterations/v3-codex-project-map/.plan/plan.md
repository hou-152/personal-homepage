## 范围

### 做

- 新建本轮 PRD 和 phase 记录。
- 重写项目卡片数据。
- 新增 Codex Workbench 分组数据和页面区块。
- 更新能力、成长线、Build Log。
- 本地构建验证。

### 不做

- 不公开私密路径和内部 ID。
- 不新增后端、CMS 或自动同步。
- 不改部署平台。
- 不把所有目录一比一搬上页面。

## 关键决策

- 使用“精选项目卡片 + Workbench 地图”的混合方案。
- 主页继续面向家人、老师、朋友和老板，不改成内部项目管理台。
- 私密项目只展示公开安全的抽象成果。
- 数据仍放在 `src/data/*.ts`，保持静态站简单稳定。
- 移动端内容宽度显式收紧，优先保证文字不被横向裁切。
